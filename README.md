# 💰 Expense Tracker

A full-stack Expense Management System built using the MERN Stack that helps users track their income and expenses, manage transactions, and analyze their financial activities through an intuitive dashboard.

## 🚀 Live Demo

### Frontend

https://expense-tracker-gules-iota.vercel.app

### Backend API

https://expense-tracker-k082.onrender.com

---

## 📌 Features

### 👤 User Authentication

* User Registration
* User Login
* Avatar Selection
* Secure Authentication

### 💳 Transaction Management

* Add Income & Expense Transactions
* Edit Transactions
* Delete Transactions
* Categorize Transactions
* Transaction Description Support

### 📊 Dashboard Analytics

* Total Balance
* Total Income
* Total Expenses
* Net Balance Calculation

### 🔍 Advanced Filtering

* Filter by Date Range
* Filter by Transaction Type
* Search Transactions
* Sort Transactions

### 📈 Reports

* Transaction Statistics
* Financial Overview
* CSV Export Support

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Shivani18105/Expense-Tracker.git
cd Expense-Tracker
```

### Install Backend Dependencies

```bash
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside backend:

```env
PORT=5000
MONGO_URL=YOUR_MONGODB_ATLAS_URL
```

---

## ▶️ Run Locally

### Backend

```bash
npm start
```

### Frontend

```bash
cd frontend
npm start
```
